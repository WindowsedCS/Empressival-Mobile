# Empressival-Mobile
此倉庫中存放著Empressival-Mobile的原始程式碼。

## 介紹
Empressival是一台可以讓您查詢特戰英豪商店優惠的Discord機器人，但我們現在不只在Discord上了，還在您的手機中，耶！

## 使用
從[此頁](https://github.com/WindowsedCS/Empressival-Mobile/releases)下載最新的APK建置並安裝。因為憑證問題，iOS系統的IPA檔案暫不提供。
